Configuration Parameters

Each flag precisely controls the scope and targeting of the intelligence gathering operation.

--country
TH
Geographic targeting for localized SERP results
--num
30
Organic results fetched per keyword query
--top-urls
3
Top-ranking URLs analyzed via Flikover
--hl
TH
Language code for search result localization
--merge
Consolidate exports into unified datasets

Integration Stack

CSV Data Columns

Keyword - Target search term
Volume - Monthly search volume
Organic Traffic - Est. traffic to URL
Position - Current ranking position
URL - Source page URL
CPC - Cost per click estimate
KD - Keyword difficulty score

Operational Advantages

Checkpoint Recovery

Built-in progress tracking allows interrupted sessions to resume exactly where they left off with --resume flag.

Smart Filtering

Automatically excludes social media domains (Facebook, Instagram, TikTok) to focus on content-rich competitors.

Rate Limit Protection

Intelligent delays between requests mimic human behavior and prevent API throttling or blocking.

Deduplication Engine

Merge process consolidates overlapping keyword data, eliminating redundancy across multiple sources.
