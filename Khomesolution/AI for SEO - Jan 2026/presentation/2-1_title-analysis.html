Processing Pipeline

Async batch processing with quality validation and checkpoint recovery.

1

Load Keywords & Resume State

Read keywords.txt and check checkpoint for previously completed/failed keywords to avoid duplication.

2

Load Dual SERP Inputs

For each keyword, load both competitions.json (top results) and keywords.json (related + PAA).

3

Build Prompt with Context

Inject SERP data into system prompt template with semantic SEO methodology instructions.

4

Call OpenRouter API

Send to Grok-4 Fast with semaphore concurrency control, rate limiting, and exponential backoff.

5

Parse & Validate YAML

Strip markdown fences, parse YAML, validate against JSON schema, compute quality score.

6

Quality Gate Check

Reject outputs below 0.7 quality score threshold. Retry with backoff if validation fails.

7

Save Output & Checkpoint

Write {keyword}-serp-analysis.yaml and update checkpoint every chunk-size keywords.
