Output Structure

Organized file hierarchy for seamless integration with downstream phases.

Directory Structure

├── {keyword-slug}/
│ ├── organic.json

├── {keyword-2}/
│ └── ...
└── master-research.json

JSON Data Fields

target_keyword - Original search term
organic[] - Top 10 ranking URLs
├── position - Ranking position
├── title - Page title
├── url - Full URL
└── description - Meta description
related_searches[] - Semantic queries
people_also_ask[] - User questions
