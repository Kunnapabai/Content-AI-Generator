Context

Information that shapes understanding

Each keyword generates 10+ context files that flow through the pipeline, building comprehensive SEO intelligence:

  • SERP Data - competitions.json (titles, snippets) + keywords.json (PAA, related)
  • Keyword Metrics - master-queries.csv with volume, traffic from Flikover
  • Title Analysis - Intent, entities, knowledge graph, pattern signals (YAML)
  • Outline + Research - H1-H4 structure → research phases → deep research findings
  • Final Output - HTML article + meta-seo.json + schema markup

Tools

External capabilities that extend functionality

We integrate 6+ specialized tools to automate the entire SEO content pipeline from research to publishing:

  • Flikover + Serper - Keyword research (volume, traffic) & SERP data (PAA, competitors)
  • Google Search Console - CTR analysis & search performance tracking
  • ChatGPT Deep Research - Manual research execution using generated prompts
  • WordPress API - Automated schema injection & content publishing
  • Google Sheets - Pipeline tracking & report exports

Model

The AI engine powering intelligence

We leverage multiple specialized AI models via OpenRouter to optimize for different tasks - balancing speed, cost, and quality.

  • Grok 4 Fast - 2M context window for deep analysis (Title, Research, Schema)
  • Gemini 3 Flash - Fast content generation with implicit caching (Outline, Meta SEO)
  • Cost-Optimized - Strategic model selection reduces costs by 60-75%

Prompt

Instructions that guide AI behavior

Our prompt library contains 10+ specialized prompts for each SEO pipeline stage. Top 5 priority prompts:

  • Title Analysis - SERP pattern extractor for intent, entities & knowledge graph
  • Title Generator - Creates 3 CTR-optimized titles with scoring algorithm
  • Outline Architect - SERP-aware H1-H4 structure with weighted signals
  • Deep Research - Converts outline to 3-7 research phases (Academic vs Industry)
  • Meta SEO - First sentence + meta description grounded in HTML context
